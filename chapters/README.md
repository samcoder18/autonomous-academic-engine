# Главы

В этой папке удобно хранить материалы по отдельным главам.

Сейчас все имеющиеся файлы в папке считаются активными:

- `00-thesis-architecture.md` - общая композиция всей работы;
- `00-introduction-framework.md` - рабочий каркас введения;
- `01-chapter-1-brief.md`, `02-chapter-2-brief.md`, `03-chapter-3-brief.md` - брифы по главам.

Минимальный набор для каждой главы:

- бриф главы;
- рабочая структура или тезисный план;
- черновик текста;
- список открытых вопросов;
- пометки после проверки.

Рекомендуемый порядок: сначала структура, затем пакет источников, затем верификация опор, потом черновик и проверка.

Архивировать бриф главы имеет смысл только после того, как соответствующая секция рукописи и лист проверки уже стабилизированы.
