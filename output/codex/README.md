# Codex Exec Runs

Здесь сохраняются финальные сообщения от thesis-ориентированных неинтерактивных запусков через [scripts/codex_thesis.sh](/Users/albina/дипломная/scripts/codex_thesis.sh), а рядом с ними - небольшие manifest-файлы с параметрами запуска.

Практический смысл папки:

- видеть, что именно сделал автономный прогон;
- видеть, с каким `preset`, `target`, моделью и search-режимом он был запущен;
- хранить краткий audit trail по типовым задачам;
- быстро сравнивать несколько прогонов по одной и той же секции.

Article-run trace живет отдельно в `articles/runs/`.

Сюда не пишется канонический текст диплома. Канонический текст thesis lane по-прежнему живет только в `manuscript/sections/`.
