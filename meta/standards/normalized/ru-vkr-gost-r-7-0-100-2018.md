# Нормализованный профиль: ГОСТ Р 7.0.100-2018 (библиографическая запись)

## Область применения

Оформление списка использованных источников в ВКР, магистерской
диссертации и кандидатской / докторской диссертации.

## Обязательные элементы записи

1. Заголовок (автор/авторы), отделяется от основной части ` / ` (slash с пробелами);
2. Основное заглавие;
3. Сведения об ответственности;
4. Сведения об издании / редакции;
5. Место и год издания / источник публикации;
6. Сведения о физической характеристике (страницы, том);
7. Канонический URL для электронного ресурса.

## Канонические правила пунктуации

- Между блоками — точка с пробелом (". ");
- Перед сведениями об ответственности — slash (`/`) с пробелами с обеих сторон;
- Электронный ресурс: `— URL: https://…` и `— Дата обращения: DD.MM.YYYY`;
- Запись завершается точкой.

## Машинные проверки

- `too-short` — запись короче 20 символов;
- `missing-terminal-period` — запись не заканчивается точкой;
- `missing-responsibility-slash` — отсутствует ` / ` между заглавием и авторами (для научных работ);
- `duplicate-canonical-url` — URL-дубликат среди записей;
- `duplicate-entry` — полный текстовый дубликат.

## Источник нормы

ГОСТ Р 7.0.100-2018. Библиографическая запись. Библиографическое описание.
