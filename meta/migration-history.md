# История миграций workspace

Ранее в репозитории хранился файл `codex-recovery.patch` (~1 МБ): единый unified diff перехода от старой модели (канон в `meta/project-canon`, manuscript в корне) к текущей многоработной схеме (`workspace.toml`, `works/<slug>/…`). Он служил только архивом миграции и не участвовал в runtime.

Актуальная модель описана в [AGENTS.md](../AGENTS.md), [workspace.toml](../workspace.toml) и [README.md](../README.md). Исторический патч из дерева исходников удалён, чтобы не раздувать клоны и не путать участников; при необходимости восстановите его из истории git по коммиту до удаления.
