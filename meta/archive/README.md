# Архив `meta/`

Здесь лежат исторические или сокращенные служебные документы, которые были выведены из активного контура после рефакторинга.

Их можно читать как справку по эволюции проекта, но в ежедневной работе опираться нужно на:

- [AGENTS.md](AGENTS.md)
- [meta/master-protocol.md](meta/master-protocol.md)
- `works/<slug>/work-canon.md` как основной канон активной работы
- [meta/project-canon.md](meta/project-canon.md) только как compatibility shim для legacy-ссылок
