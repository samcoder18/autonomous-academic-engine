# Source Package — Проблемы реализации конституционных поправок

Дата сборки: 14.06.2026. Последняя проверка динамического нормативного блока: 15.06.2026.

## Границы пакета

Пакет подтверждает юридические факты принятия и реализации поправок 2020 года, а также дает доктринальную основу для оценки правовой определенности, разделения властей, федерализма и самостоятельности местного самоуправления. Он не используется для партийно-политических оценок и не подменяет анализ официальных текстов сообщениями СМИ.

## Правила использования

- Сильные утверждения о содержании поправок и последующих законов связываются с первичным актом.
- Доктрина используется для классификации и объяснения, а не вместо официального текста.
- При отсутствии проверенного экземпляра не приводятся дословные цитаты и номера страниц.
- Внешняя критика Венецианской комиссии представляется как позиция международного экспертного органа, а не как обязательная для России оценка.
- Выводы о рисках централизации, правовой неопределенности и исполнимости социальных гарантий маркируются как аналитические.
