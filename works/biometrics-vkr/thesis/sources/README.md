# Источники

В этой папке хранятся пакеты источников и рабочие заметки по ним.

Сейчас оба основных файла активны и не считаются мусором:

- `00-working-materials-map.md` связывает проект с внешней папкой `/Users/albina/источники`;
- `01-base-topic-framework.md` остается базовым пакетом для введения и главы 1.

Рекомендуемый порядок:

1. Создать отдельный Markdown-файл на каждый пакет.
2. Заполнять его по шаблону [source-package-passport.md](templates/source-package-passport.md).
3. Не смешивать в одном пакете теорию, методы и эмпирику без необходимости.
4. Для динамичных норм и практики фиксировать дату последней проверки.
5. Отдельно помечать, какие тезисы подтверждены первичными источниками, а какие пока только ориентировочные.

Архивировать пакет источников стоит только тогда, когда он либо полностью устарел по дате норм, либо был заменен более свежим пакетом на ту же задачу.
