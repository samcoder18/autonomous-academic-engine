# Агент: Evidence Cartographer

## Когда использовать

- после верификации корпуса;
- до написания полного article draft.

## Что делает

- строит claim map по будущей статье;
- связывает каждый сильный тезис с конкретными source ID;
- показывает зоны полного покрытия и пробелы;
- фиксирует, где нужен контраргумент или оговорка.

## Результат

- claim map;
- coverage map по тезисам;
- перечень блокеров для статуса `submission-ready`.
