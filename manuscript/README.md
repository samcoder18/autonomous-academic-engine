# Рукопись thesis lane

Здесь хранится единый черновик thesis lane.

## Как устроено

- `sections/` — канонические текстовые секции.
- `full-draft.md` — автоматически собранный полный документ.

## Правило

Не редактировать `full-draft.md` как основной рабочий файл. Изменения вносятся в секции, после чего полный документ пересобирается скриптом.
